### `p`

```html

    p{/* 1. html태그를 선택자로 사용할 수 있다. */
    	color: #03a8be;
        background-color: yellow;
    }
</style>
```

### `.class`

```html
<style type="text/css">
    .myclass{/* 2. 태그 내부에 class 속성을 정의하고 선택자로 사용할 수 있다.
                            => 중복해서 정의하는 것이 가능 */
                color: blue;
    }
</style>
```
